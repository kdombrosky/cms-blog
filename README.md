# Tech CMS Blog

## Description 
This site functions as a host for tech bloggers, where users can perform most of the basic necessary functions for an operational social media: 
  - Create accounts
  - Share blog posts, thoughts, opinions
  - Make comments on posts 
  - Update/Delete posts 
  - Save login sessions
